# Lucimar Costuras — Backend

Repositório backend do site **Lucimar Costuras**, que funcionará como **portfólio e vitrine do trabalho** da costureira Lucimar.

Este sistema será responsável por gerenciar as informações expostas no frontend, como peças produzidas, descrições, imagens e dados de contato.

## Tecnologias Utilizadas

- **Java**
- **Spring Boot**
- **API RESTful**
- **Docker**
